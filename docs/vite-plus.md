# Vite+ Monorepo Starter

A starter for creating a Vite+ monorepo.

## Install

install vite-plus bia brew

```shell
brew install vite-plus
vp env off
```

## Development

- Check everything is ready:

```shell
vp run ready
```

- Run the tests:

```shell
vp run -r test
```

- Build the monorepo:

```shell
vp run -r build
```

- Run the development server:

```shell
vp run dev
```

- Other commands

```shell
vp run fix
vp run check
vp run clean
```
