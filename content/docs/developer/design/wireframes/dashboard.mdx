Modern SaaS starters

## Recommended Stack

### Frontend

- TanStack Start
- TanStack Router
- TanStack Query
- TanStack Table
- TanStack Form
- React 19
- Tailwind v4
- shadcn/ui
- Base UI (base-nova)
- Paraglide JS

### Backend

- TanStack Start server functions
- Better Auth
- Better Auth Organizations Plugin
- Drizzle ORM
- PostgreSQL

### State

- Zustand only for UI state

  - sidebar collapsed
  - command palette
  - table preferences
  - theme
  - active organization cache

Everything else should live in TanStack Query.
