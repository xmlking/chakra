## Overview

## Installation

```shell
bun x @fumadocs/cli
```

## Usage

```shell
# Select and install components.
bun x @fumadocs/cli add
# A simple way to customize Fumadocs layouts.
bun x @fumadocs/cli customise
# Generate files tree
bun x @fumadocs/cli tree ./my-dir ./output.mdx
```
